What is Digitron Associates?

Digitron Associates is a real-world e-commerce web application built for a CCTV and security solutions business based in Hubbali, Karnataka. Tech Stack Decision Rationale

Technology Choice Architectural Motivation & Rationale
React + Vite Delivers rapid Hot Module Replacement (HMR), components encapsulation, and efficient DOM renders.
Express.js Provides a lightweight, unopinionated routing layer to construct API responses rapidly.
MongoDB Flexible document model seamlessly maps varied specifications of diverse security equipment.
Prisma ORM Type-safe query compilation, simple schema files, and automated database transitions.
React Query Performs declarative background caching and avoids redundant client network fetch loops.
Stateless JWT Enables lightweight validation headers without demanding storage access operations on the server.
Google OAuth 2.0 Provides quick sign-in capabilities, removing purchase friction for CCTV customers.

Featured Code Snippet

A look at the custom multi-filter solution powering the product catalog:

ProductFilterEngine.js 
// Custom multi-criteria evaluation routine (optimized for 400+ SKUs)
const filteredProducts = useMemo(() => {
  return products.filter(product => {
    const matchesCategory = !filters.category || product.category === filters.category;
    const matchesBrand    = !filters.brand    || product.brand === filters.brand;
    const matchesType     = !filters.type     || product.type === filters.type;
    const matchesResolution = !filters.res    || product.resolution === filters.res;
    
    return matchesCategory && matchesBrand && matchesType && matchesResolution;
  });
}, [products, filters]);

Defect Management Log

🐞

Filter State Retention Defect

Bug: Active filter parameters were not resetting when switching categories, displaying blank cells.
Solution: Embedded cleanup dispatch handlers inside React's `useEffect` hooks.

🐞

MongoDB Cold-Start Failure

Bug: Server database connections would periodically timeout on cold environment starts.
Solution: Appended optimized `serverSelectionTimeoutMS` variables inside database connection configurations.

🐞

JWT Expiry Unhandled Crash

Bug: Admin authentication tokens expiring without notice caused raw network failure messages.
Solution: Programmed Axios token interceptors executing an automatic routing to the login interface upon 401 returns.

Retrospective

Key Reflections

Building a complete project case study was one of the most challenging and rewarding parts of my engineering journey so far. It taught me how to think beyond code and communicate decisions clearly.

01

🤝 Collaborative Coding beats Solo Dev

Working in a team of five (Goureesh, Ashutosh, Yashwant, Anoosha, and myself) taught me the value of delegation, alignment, and source control. Building different modules of the CCTV platform required clear interfaces and mutual trust.

"Software engineering is a team sport; code is only as clean as the team's communication."

02

🔌 Bridging Frontend & Backend Logic

Connecting React state flows to an Express backend was a massive learning loop. Understanding how database queries, middleware security gates, and REST controllers interact has shifted my perspective to system-level thinking.

"An interface is just a window; the real magic happens in the data pipeline behind it."

03

💾 Seeding Workflows & Database Sync

Managing product seeding files containing over 400 lines of CCTV catalog components forced me to learn schema normalization, migration safety, and how to verify backend database connections early on.

"Design schemas for tomorrow's queries, but parse files for today's database seeds."

04

⏱️ Agile Sprints & Incremental Slices

Splitting our work into Sprints made the project highly manageable. We learned to ship incremental working features, debug them in parallel, and continuously review progress together during team standups.

"Sprints divide complex backlogs into measurable checkpoints of working software."
